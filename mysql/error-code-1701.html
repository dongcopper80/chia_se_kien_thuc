When you run MySQL script like :
TRUNCATE `sys_role`;
If table `sys_role` have referenced constraint with others table then you get error:
TRUNCATE `sys_role` Error Code: 1701. Cannot truncate a table referenced in a foreign key constraint (`goingo.com`.`sys_role_auth`, CONSTRAINT `FK8cdtxc5xi6h8lbah6bx0aikfi`)

In this case you have 2 options:
Option 1:

  1. 1. Remove constraints
  2. 2. Perform TRUNCATE
  3. 3. Delete manually the rows that now have references to nowhere
  4. 4. Create constraints

Option 2:
You can run this script block
SET FOREIGN_KEY_CHECKS = 0;
TRUNCATE sys_role;
SET FOREIGN_KEY_CHECKS = 1;

It's done !!!!!!!!!!!!!!!!! This problem is solved.
